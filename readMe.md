## 1. 목표와 기능
목표
* Python DRF 의 CRUD 기능을 구현할 수 있는 페이지를 만들고자 하였다
기능
* 제목 / 내용의 입력으로 DB에 권한 제한 없이 메모를 입력할 수 있는 웹페이지를 작성하였다
* DRF의 지원을 받아 주고 받는 Json 데이터의 흐름을 확인하고 Django가 처리하는 로그를 확인할 수 있었다
